#❓ How to use it

```bash

git clone https://github.com/arthur-lage/recipes.git

cd recipes

yarn
# if you don't have yarn installed, you can use npm install. 
npm install

yarn start
# if you don't have yarn installed, you can use npm start. 
npm start

```
